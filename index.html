Gesture-Controlled Car

Designed and built a hand-gesture controlled robotic car using dual Arduino boards and RF communication. The system used an ADXL335 accelerometer mounted on a wearable glove to detect real-time hand motion and orientation, which was processed by a transmitting Arduino to convert gestures into directional commands.

These signals were sent wirelessly via an RF transmitter module to a second Arduino receiver module mounted on the vehicle. The receiver translated the incoming signals into motor control instructions, using an L293D motor driver to independently control the movement of the vehicle's tires ‚Äî supporting forward, reverse, left, right, and stop commands based on wrist tilt and gesture angle.

This project demonstrates mastery of serial communication, motion sensing, real-time signal processing, and motor control. It was a successful implementation of wearable-based gesture control and wireless actuation.

Technologies Used: Arduino Uno, RF Transmitter/Receiver, ADXL335 Accelerometer, L293D Motor Driver, C/C++, Embedded Systems, Motion Control

Smart Irrigation System

Developed an automated smart irrigation system using an ESP32 and a soil moisture sensor to optimize water usage based on soil conditions. When soil moisture dropped below a defined threshold, the system automatically triggered a motor to activate water flow through the irrigation pipeline.

To estimate daily water usage, I calculated the approximate volume of water delivered based on pipe diameter and irrigation duration. This data was then packaged and sent to a remote MySQL database using HTTP POST requests from the ESP32, enabling real-time water usage monitoring.

The solution supports scalable deployment in agricultural zones and showcases effective integration of sensor-actuator control, data estimation logic, and cloud communication for precision irrigation.

Technologies Used: ESP32, Soil Moisture Sensor, Relay + Motor Driver, Pipe Flow Calculations, MySQL, HTTP POST, Embedded C++, IoT, Cloud Logging

Temperature-Controlled Water Heater

Built a smart temperature-controlled water heater system using an Arduino Uno and DS18B20 temperature sensor to monitor water temperature in real time. The system allowed users to set their desired temperature using push-button inputs, which was displayed and updated on a connected 16x2 LCD screen.

The system continuously compared the real-time sensor reading with the user-defined target temperature. When the temperature exceeded safe thresholds or deviated beyond the tolerance range, an alarm buzzer was triggered to notify the user.

It also included fail-safes for overheat protection and button debounce handling for accurate input selection. This project highlights key embedded concepts such as real-time data acquisition, actuator control, user interaction, and safety automation.

Technologies Used: Arduino Uno, DS18B20 Temperature Sensor, 16x2 LCD Display, Buzzer Alarm, Push Buttons, Embedded C, Sensor Calibration
